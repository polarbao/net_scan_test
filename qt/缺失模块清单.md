# Qt项目缺失模块清单

## 一、已创建的模块
1. ✅ MainWindow - 主窗口
2. ✅ PrintThread - 打印线程
3. ✅ ParameterManager - 参数管理

## 二、缺失的模块

### 2.1 线程模块
- [ ] MonitorThread - 监控线程（对应MFC的MonitorThread静态函数）

### 2.2 对话框模块
- [ ] PrintJobDialog - 打印作业对话框（对应CPrtJobDlg）
- [ ] SysParamDialog - 系统参数对话框（对应CSysParamDlg）
- [ ] ImgLayerSetDialog - 图像图层设置对话框（对应CImgLayerSetDlg）
- [ ] VTSetDialog - VT设置对话框（对应CVTSetDlg）
- [ ] AdjParamDialog - 调整参数对话框（对应CAdjParamDlg）
- [ ] ReloadWaveDialog - 重载波形对话框（对应CReloadWaveDlg）
- [ ] MoveCtlDialog - 运动控制对话框（对应CMoveCtlDlg，可选）
- [ ] AirInkDialog - 气墨控制对话框（对应CAirInkDialog）
- [ ] AdibCtrlDialog - ADIB控制对话框（对应CAdibCtrlDlg）
- [ ] LackJetDialog - 缺喷检测对话框（对应CLackJetDlg）

### 2.3 自定义控件模块
- [ ] ImagePreviewWidget - 图像预览控件（对应CPicScreen，已有框架需完善）
- [ ] PrintInfoTreeWidget - 树形列表控件（对应CSWListTreeCtrl）
- [ ] InkStateWidget - 墨水状态控件（对应CInkStateCtrl）
- [ ] ImportListWidget - 导入列表控件（对应CImportListCtrl）

### 2.4 应用程序入口
- [ ] main.cpp - 应用程序入口

### 2.5 工具函数
- [ ] ImageLoader - 图像加载工具类（对应GetSrcData函数）

## 三、优先级

### 高优先级（核心功能）
1. MonitorThread - 设备监控必需
2. PrintJobDialog - 打印作业配置必需
3. ImagePreviewWidget - 图像预览必需
4. PrintInfoTreeWidget - 信息显示必需
5. main.cpp - 应用程序入口必需

### 中优先级（重要功能）
6. SysParamDialog - 系统参数配置
7. VTSetDialog - VT参数设置
8. AdjParamDialog - 调整参数

### 低优先级（辅助功能）
9. ImgLayerSetDialog - 图层设置
10. ReloadWaveDialog - 波形重载
11. MoveCtlDialog - 运动控制（可选）
12. AirInkDialog - 气墨控制
13. AdibCtrlDialog - ADIB控制
14. LackJetDialog - 缺喷检测
15. InkStateWidget - 墨水状态显示
16. ImportListWidget - 导入列表
